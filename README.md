****Mini Code Editor****

This repository contains a Mini Online Code Editor built using HTML, CSS, and JavaScript.
It allows users to write and test HTML, CSS, and JavaScript code in real-time.

***Features***

Live Preview inside an iframe

Tabs for HTML, CSS, and JS

Run Code button to execute user-written code

Simple and clean UI

Beginner-friendly structure


 ***Technologies Used***

HTML5

CSS3

JavaScript (Vanilla JS)

***How It Works***

User writes or edits HTML, CSS, or JS in separate textareas

The Run Code button injects the code into an iframe

Output is displayed instantly
